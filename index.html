Element Description Link
table Represents tabular data that is, information presented in a two-dimensional table comprised of rows and columns of cells containing data. Link
tbody Encapsulates a set of table rows (tr elements), indicating that they comprise the body of the table Link
td Defines a cell of a table that contains data. It participates in the table model. Link
th Defines a cell as header of a group of table cells. The exact nature of this group is defined by the scope and headers attributes. Link
tfoot Defines a set of rows summarizing the columns of the table. Link
